# Prequisites

- Python 3
- YFinance
- React
- Chart.js

 # Running Locally

1. Run the following `pip` and `npm`command to install the dependencies:
   ```
   pip install -r requirements.txt
   ```
   ```
   npm install 
   ```

2. Run the following commands to local host:
   
   ```
   uvicorn main:app --reload

   ```
   ```
   npm start
   ```
